## Запуск тестов

1. Установка всех зависимостей 
```
npm install
```

2. Запуск тестов 
```
npx playwright test
```

3. Отоброжение отчета после прогона тестов
```
npx playwright show-report
```

4. Запуск тестов в UI режиме 
```
npx playwright test --ui
```
